Activity: JS Graffiti

In this activity, we challenge you to use JavaScript to deface an HTML page as it's displayed in your browser, using JavaScript.

Select elements and change their styles; give them goofy colors, borders, and even -dare we suggest it- transformations.

The only rule is that you cannot edit the HTML or CSS inside index.html

Here are some steps to think through if you get stuck:

  1. Look at the page in your browser.
  2. Think of something you would like to change visually.
  3. Next, look at the html in your code editor.
  4. Find an attribute you can use to select the element(s) you want to change.
  5. Select that element in your JavaScript file, using querySelector or querySelectorAll. Use console.log to make sure you have selected the element(s) you want.
  6. Figure out what CSS property you want to use to make the change you're going for. You can use this resource to search for applicable styling rules.
